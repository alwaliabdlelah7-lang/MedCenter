# MedCenter HIS (Hospital Information System)

نظام متطور لإدارة معلومات المستشفيات مبني باستخدام أحدث التقنيات لتقديم تجربة مستخدم سلسة وتفاعلية.

## 🚀 المميزات
- **إدارة المرضى**: سجلات طبية كاملة، إدارة الزيارات، والتاريخ المرضي.
- **إدارة العيادات**: تنظيم الطوابير، التخصصات، والمواعيد.
- **نظام المختبر**: طلب الفحوصات، إدخال النتائج، وإصدار تقارير طبية احترافية.
- **التشخيص المدعوم بالذكاء الاصطناعي**: نظام ذكي للمساعدة في تحليل البيانات الطبية.
- **PWA (تطبيق متعدد المنصات)**: يمكن تثبيته على الأندرويد، ويندوز، والأجهزة اللوحية.
- **تصميم عصري**: واجهة مستخدم زجاجية (Glassmorphism) مريحة للعين.

## 🛠 التقنيات المستخدمة
- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS.
- **Animation**: Motion (Framer Motion).
- **Backend/Database**: Firebase (Firestore & Auth).
- **AI**: Gemini Pro API.

## 📱 التشغيل على المنصات
- **الويب**: متوافق مع كافة المتصفحات الحديثة.
- **أندرويد/لوحي**: يمكن تثبيت التطبيق مباشرة من المتصفح عبر خاصية (Add to Home Screen).
- **ويندوز/ديسكتاب**: يمكن تثبيته كتطبيق مستقل عبر المتصفح.

## 📦 البدء بالعمل
1. قم بتحميل المستودع.
2. قم بتثبيت المكتبات: `npm install`
3. قم بإنشاء ملف `.env` وإضافة مفتاح Gemini: `GEMINI_API_KEY=your_key`
4. ابدأ التشغيل: `npm run dev`
