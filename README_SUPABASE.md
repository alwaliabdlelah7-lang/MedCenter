# HIS Supabase Setup Guide

لضمان عمل النظام بشكل كامل ودقيق مع Supabase، يرجى اتباع الخطوات التالية:

## 1. إعداد قاعدة البيانات (SQL)
قم بنسخ محتوى ملف `supabase_schema.sql` الموجود في جذر المشروع ولصقه في **SQL Editor** داخل لوحة تحكم Supabase الخاصة بك، ثم اضغط على **Run**. هذا سيقوم بإنشاء كافة الجداول (المرضى، الأطباء، الصيدلية، إلخ) مع الصلاحيات الأساسية.

## 2. مفاتيح الربط (Environment Variables)
تأكد من إضافة المفاتيح التالية في إعدادات البيئة الخاصة بك (أو ملف `.env` محلياً):
- `VITE_SUPABASE_URL`: رابط المشروع (Project URL).
- `VITE_SUPABASE_ANON_KEY`: مفتاح الواجهة العام (Anon Key).

## 3. زرع البيانات الأولية (Initial Seeding)
بمجرد تشغيل التطبيق:
1. اذهب إلى **الإعدادات** (Settings).
2. اختر تبويب **البيانات السحابية (Supabase)**.
3. اضغط على زر **تزويد السحابة بالبيانات المرجعية**.
سيقوم النظام تلقائياً برفع قائمة الأدوية، تخصصات الأطباء، الخدمات الطبية، والمختبرات الأولية لتبدأ العمل فوراً.

## 4. نظام المستخدمين
يمكنك تسجيل الدخول للمرة الأولى باستخدام:
- اسم المستخدم: `admin`
- كلمة المرور: `123` (أو اتركها فارغة إذا لم تكن قد هيأت Auth بعد).
