# دليل بناء تطبيق الأندرويد (Android Build Guide)

لقد تم إعداد النظام بالكامل لبناء تطبيق الأندرويد الخاص بك تلقائياً عبر GitHub أو يدوياً في بيئتك المحلية.

## 1. البناء التلقائي عبر GitHub Actions
بمجرد رفع (Push) الكود إلى GitHub، سيقوم النظام تلقائياً بالآتي:
* تثبيت Java و Node.js.
* بناء ملفات الويب.
* تحويل المشروع إلى أندرويد.
* استخراج ملف **APK** جاهز للتثبيت.

**كيفية الحصول على ملف APK:**
1. اذهب إلى تبويب **Actions** في مستودع GitHub الخاص بك.
2. اختر آخر عملية بناء ناجحة باسم "Build & Deploy Android App".
3. ستجد ملف الـ APK في قسم **Artifacts** أسفل الصفحة.

---

## 2. البناء اليدوي (Local Build)
إذا كنت تريد البناء على جهازك الخاص، اتبع الخطوات التالية:

### المتطلبات الأساسية:
* **Node.js** (إصدار 18 أو أعلى).
* **Android Studio**.
* **Java JDK 17**.

### الأوامر:
لقد قمت بإنشاء ملف `build-android.sh` ليسهل عليك العملية. يمكنك تشغيله عبر:
```bash
bash build-android.sh
```

أو تنفيذ الأوامر يدوياً:
```bash
# تثبيت المكتبات
npm install

# بناء مشروع الويب
npm run build

# مزامنة الملفات مع الأندرويد
npx cap sync android

# فتح المشروع في Android Studio
npx cap open android
```

---

## 3. إعدادات النشر (Production)
عندما تصبح جاهزاً لنشر التطبيق على متجر Google Play:
1. يجب عليك إنشاء "Key Store" لتوقيع التطبيق.
2. استخدام الأمر `./gradlew assembleRelease` بدلاً من `assembleDebug`.

تأكد من تحديث ملف `capacitor.config.ts` باسم الحزمة (App ID) الذي تريده قبل النشر النهائي.
