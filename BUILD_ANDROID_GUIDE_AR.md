# دليل بناء تطبيق الأندرويد (Android Build Guide)

لقد تم إعداد النظام بالكامل لبناء تطبيق الأندرويد الخاص بك تلقائياً عبر GitHub أو يدوياً في بيئتك المحلية.

## 1. البناء التلقائي عبر GitHub Actions
بمجرد رفع (Push) الكود إلى مستودع GitHub الخاص بك، سيبدأ البناء التلقائي:
* **تثبيت المكاتب**: يتم تحميل كافة الأدوات اللازمة (Node.js, Java).
* **بناء الويب**: يتم تحويل كود React إلى ملفات جاهزة.
* **تحويل الأندرويد**: يتم دمج ملفات الويب داخل مشروع أندرويد.
* **استخراج APK**: يتم إنتاج ملف APK قابل للتثبيت.

**كيفية الحصول على ملف APK:**
1. اذهب إلى تبويب **Actions** في GitHub.
2. ستجد عملية باسم "Build Android APK".
3. بعد الانتهاء، ستجد ملف **MedCenter-HIS-Debug-APK** في قسم **Artifacts**.

---

## 2. البناء اليدوي (Local Build)
استخدم السكريبت المطور الذي قمت بإنشائه لك:
```bash
bash build-android.sh
```

هذا السكريبت يقوم بـ:
1. التأكد من وجود كافة المكاتب.
2. بناء مشروع الويب.
3. إضافة منصة الأندرويد إذا لم تكن موجودة.
4. مزامنة الأكواد.
5. بناء ملف APK محلياً.

### المتطلبات:
* **Node.js 20+**
* **Android Studio**
* **Java JDK 17**

---

## 3. ملاحظات هامة
* **الصلاحيات**: التطبيق يحتاج صلاحية الكاميرا (لقراءة الـ QR) والميكروفون؛ تأكد من قبولها عند التشغيل.
* **قاعدة البيانات**: التطبيق مرتبط بـ Firebase؛ تأكد من أن ملف `firebase-applet-config.json` يحتوي على الإعدادات الصحيحة.
* **التوقيع**: ملف الـ APK الناتج هو "Debug"؛ للنشر على المتجر تحتاج إلى "Signed Release APK".
